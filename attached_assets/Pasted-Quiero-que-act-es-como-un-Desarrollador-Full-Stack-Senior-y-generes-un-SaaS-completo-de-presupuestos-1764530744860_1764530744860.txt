Quiero que actúes como un Desarrollador Full-Stack Senior y generes un SaaS completo de presupuestos de transporte.

OBJETIVO:
Crear una aplicación SaaS donde el usuario introduzca origen, destino y tipo de vehículo, y el sistema calcule automáticamente el precio del transporte usando mis reglas de precios y la distancia real obtenida con la API gratuita de OpenRouteService.

FUNCIONALIDADES CORE:
1. Login y autenticación de usuarios (Replit Auth).
2. Panel Administrativo para gestionar:

   - Precio base
   - Precio por kilómetro
   - Recargo por peajes
   - Precio mínimo
   - Tipos de vehículo
3. Formulario para que el cliente introduzca:
   - Dirección de origen
   - Dirección de destino
   - Tipo de vehículo
4. Cálculo AUTOMÁTICO de distancia usando la API de OpenRouteService.
5. Selección automática de zona según los kilómetros y el país.
6. Cálculo del presupuesto siguiendo estas reglas oficiales:

7. Cálculo del precio final incluyendo:
   - Base
   - Coste por km
   - Peajes
   - Extras opcionales
   - Total final

8. Mostrar desglose al cliente.
9. Panel de usuario con historial de presupuestos generados.

- Base de datos PostgreSQL integrada de Replit

- Todo funcionando sin servicios de pago
